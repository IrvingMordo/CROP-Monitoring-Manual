# EOSDA Crop Monitoring - Manual de Usuario

Este repositorio contiene el manual de usuario detallado para la plataforma **EOSDA Crop Monitoring**. 

## Contenido
- [manual_usuario.md](manual_usuario.md): Guía completa, técnica y operativa sobre el uso de la plataforma, incluyendo:
    - Procedimientos de inicio y cierre de sesión.
    - Diccionario de índices de vegetación.
    - Flujos de trabajo de scouting y mapas VRA.
    - Análisis avanzado de clima y suelo.

## Cómo usar este manual
Este manual está escrito en formato **Markdown**. Se recomienda usar un editor compatible con este formato (como VS Code, Obsidian o la propia interfaz de GitHub) para una experiencia óptima de lectura.
